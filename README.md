## English

### What is this?

Memeify-Thumbnail is an open-source project designed to inject custom "overlay" characters onto video thumbnails. Tired of the standard YouTube thumbnails? Want to put your favorite character on Bilibili or Niconico video covers? There are the tools for you.

### Why does it exist?

We are taking the core concept from [MrBeastify-Youtube](https://www.google.com/search?q=https://github.com/MagicJinn/MrBeastify-Youtube) and evolving it. While the original project was a simple browser extension limited to YouTube, our goal is to build a universal "Overlay Engine" that supports more characters, multiple platforms and devices, including mobile apps via Magisk/LSPosed modules.

### Core Features

* **Platform Agnostic:** Targeting YouTube, Bilibili, and Niconico.
* **Customizable:** Swap characters as you please.
* **Cross-Platform:** Browser extensions for desktop, and system-level injection (Magisk/LSPosed) for mobile.

### Get Involved

This project is essentially just "pasting character cutouts onto thumbnails." We believe the fun lies in the community-driven "Character-Packs."
